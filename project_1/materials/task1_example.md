# Шаблон оформления для Задания 1

Это шаблон — он показывает только **ФОРМАТ**. Конкретные кейсы и обоснования — в `readme.md`. **Не копируй — заполняй самостоятельно!**

---

## Как оформить каждый кейс

Для каждого из 10 кейсов структура одинакова:

### Кейс N: [Текст вопроса]

**Тип аналитики:** Descriptive / Diagnostic / Predictive / Prescriptive

**Обоснование:** Краткое объяснение (2–3 предложения), почему выбран именно этот тип.

**Какие данные потребуются:**
- Столбец 1
- Столбец 2
- ...

---

## Пример на нейтральном кейсе

### Кейс 0 (демо): Сколько чашек кофе было продано в кофейне за прошлый месяц?

**Тип аналитики:** Descriptive (Описательная)

**Обоснование:** Вопрос задан в прошедшем времени и требует агрегации уже имеющихся данных о продажах. Мы не ищем причину и не строим прогноз — просто описываем факт. Это классический пример описательной аналитики.

**Какие данные потребуются:**
- Дата транзакции
- Тип товара (чашка кофе / другое)
- Количество единиц в чеке
- Статус заказа (выполнен / отменён)

---

*Пример выше — только для понимания формата. Твоя задача — разобрать 10 кейсов из `readme.md`, связанных с клиникой «Здоровье+».*
