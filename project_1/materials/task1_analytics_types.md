Задание 1. Линзы аналитика
Что нужно знать?
Для решения данного задания нужны знания в следующих темах:
Типы аналитики: Описательная (Descriptive), Диагностическая (Diagnostic), Предиктивная (Predictive), Предписывающая (Prescriptive).
Жизненный цикл данных: Сбор, хранение, очистка, анализ, интерпретация.
Бизнес-метрики в медицине: Загрузка (utilization), выручка (revenue), процент отмен (no-show rate).
Контекст
Представь, что ты сидишь на большой планерке руководителей сети клиник «Здоровье+». Директора разных департаментов один за другим озвучивают проблемы, которые мешают бизнесу расти. Твоя задача как аналитика — не просто выслушать их, а перевести «боли» бизнеса на язык данных и определить, какой аналитический подход поможет найти ответ. Правильный выбор типа аналитики на старте экономит недели работы и позволяет использовать именно те инструменты, которые дадут результат.
Задача
Классифицировать 10 представленных ниже кейсов по типам аналитики и подготовить аргументацию для руководства.
Список кейсов для анализа:
Какова средняя загрузка терапевтов во всех филиалах за прошлый месяц?
Почему в отделении неврологии выручка упала на 15% при сохранении того же количества записей?
Какое количество пациентов с симптомами гриппа обратится в клинику в следующем месяце?
Как составить график работы персонала так, чтобы минимизировать время ожидания пациентов в утренние часы пик?
Каков текущий процент отмен (No-Show) записей к стоматологам?
Есть ли зависимость между стажем врача и количеством повторных обращений пациентов?
С какой вероятностью конкретный пациент пропустит прием на основе его истории посещений?
Как автоматически перераспределить поток пациентов между филиалами, чтобы устранить очереди на МРТ?
Какова динамика цен на расходные материалы для лаборатории за последний год?
Почему пациенты из определенного района города стали реже записываться на первичный прием?
Действие
Создай в папке src файл task1_analytics_types.md. Для каждого из 10 кейсов:
Укажи тип аналитики.
Напиши обоснование (2-3 предложения), почему выбран именно этот тип.
Перечисли, какие конкретно данные тебе потребуются для ответа на этот вопрос (например: дата визита, статус визита, ФИО врача и т.д.).
Результат
В итоге загрузи в репозиторий файл с аргументацией: task1_analytics_types.md.
